## Angularは優れているのか

AngularはSPAを作るためのフレームワーク。同じ目的を持ったフレームワークとしてReactやVueが有名。

おそらく一番人気なSPAフレームワークはReactだと。でも個人的にAngularが好きだから使ってるだけ。言語もフレームワークも好きなのを使えば良いのさ。
